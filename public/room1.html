Smart Energy Limiter - 1 Room Wiring Documentation

Arduino UNO Connections
  • ACS712 current sensor:
    • VCC → 5V
    • GND → GND
    • OUT → A0
  • Relay Module:
    • IN → D8 (relayPin)
    • VCC → 5V
    • GND → GND
  • ESP32 Control Signal:
    • D7 → ESP32 GPIO23
  • Serial Connection:
    • TX (Pin 1) → ESP32 RX2 (GPIO16)
    • GND → ESP32 GND
  • LCD (I2C 0x27) (optional):
    • VCC → 5V
    • GND → GND
    • SDA → A4
    • SCL → A5
ESP32 Connections
  • WiFi Server:
    • Handles dashboard at http://[IP]/
  • Arduino Communication:
    • RX2 (GPIO16) ← Arduino TX
    • TX2 (GPIO17) (optional)
    • GND ↔ Arduino GND
  • Manual Override Control:
    • GPIO23 → Arduino D7
    • LOW = Normal mode (Auto)
    • HIGH = Manual override (Relay OFF)
  • Power:
    • 5V from USB or external 5V
    • GND shared with Arduino
Relay → Load Wiring (1 Room)
  • Relay COM → +5V (Power Source)
  • Relay NO → Load + (LED / Fan)
  • Load − → GND (Power Source)
  • When Relay ON: COM ↔ NO → Load is powered
  • When Relay OFF: COM ↔ NO open → Load off
ACS712 Wiring (Current Sensor)
  • Place ACS712 in series with +5V line feeding load.
  • IN terminal → +5V Supply
  • OUT terminal → Relay COM pin
  • It measures total load current drawn by relay output.
Power Supply
  • +5V → Arduino + ESP32 + Relay + ACS712
  • GND → Common Ground (All devices must share GND)
  • Recommended: 5V 2A regulated power supply
