Smart Energy Limiter Wiring Guide

ACS712 → Arduino UNO
  • VCC → 5V
  • GND → GND
  • OUT → A0
  • Measure +5V line feeding relays
Arduino UNO → ESP32 (Serial)
  • TX (Pin 1) → RX2 (GPIO16)
  • RX (Pin 0) → TX2 (GPIO17)
  • GND → GND
ESP32 → Relays (Room Controls)
  • GPIO25 → Relay 1 (Room 1)
  • GPIO26 → Relay 2 (Room 2)
  • GPIO27 → Relay 3 (Room 3)
  • VCC → 5V
  • GND → GND
Relay → Loads
  • COM → +5V
  • NO → Load +
  • Load − → GND
Power Supply
  • +5V → Arduino + ESP32 + Relays + ACS712
  • GND → Common ground
